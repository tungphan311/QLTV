# Tính năng chính:

- Đăng nhập tài khoản 

- Lập thẻ độc giả

- Tiếp nhận sách mới

- Tra cứu sách

- Lập phiếu mượn sách

- Lập phiếu trả sách

- Báo cáo, thống kê tình hình mượn sách

- Thay đổi quy định
